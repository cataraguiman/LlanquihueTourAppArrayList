## 📘 Descripción general del sistema

Este proyecto corresponde a la actividad formativa de la semana 4: Usando colecciones provistas por el lenguaje Java. Esta actividad consiste en la lectura de datos desde un archivo .txt para instanciar objetos que serán almacenados en una colección del tipo ArrayList (aplicando operaciones sobre la colección como: recorrido, búsqueda y filtrado). El requerimiento es solicitado por la agencia de turismo Llanquihue Tour, la cual reqyuere gestionar de manera eficiente la información de sus tours, clasificados por tipo, con el objetivo de mostrarlo en catálogos según la preferencia de cada uno de sus clientes.
